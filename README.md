# Minesweeper

A browser-based Minesweeper game built using HTML, CSS, and JavaScript.

## Features
- Multiple difficulty levels (Easy, Intermediate, Expert)
- First-click safe system
- Flagging system 🚩
- Timer ⏱️
- Light/Dark mode 🌙
- Instructions popup
- Particle background effects ✨

## How to Play
- Left-click to reveal a tile
- Right-click to place/remove a flag
- Numbers indicate nearby mines
- Reveal all safe tiles to win

## Controls
- Press **R** to restart the game

## Tech Used
- HTML
- CSS
- JavaScript
